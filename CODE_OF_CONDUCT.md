# Code of Conduct

This is a small, personal repo — field notes and a build path, maintained by one operator. The rules are short on purpose.

- **Be concrete, not hostile.** Disagree with a claim by pointing at what's wrong and why, not by attacking the person making it. That matches the repo's own style (see [CONTRIBUTING](CONTRIBUTING.md)).
- **No harassment, no doxxing, no hate speech.** Ordinary baseline — issues and PRs that violate it get closed and the author blocked, no debate.
- **Off-topic drama gets closed.** This repo is about the three pillars. Take unrelated arguments elsewhere.

Report a problem by opening an issue, or by contacting [@KeilerHirsch](https://github.com/KeilerHirsch) directly for anything you'd rather not post publicly.

## Deutsch

Das hier ist ein kleines, persönliches Repo — Feldnotizen und ein Bau-Pfad, gepflegt von einem Operator. Die Regeln sind bewusst kurz.

- **Konkret, nicht feindselig.** Widersprich einer Behauptung, indem du zeigst was falsch ist und warum — nicht indem du die Person angreifst, die sie macht. Passt zum eigenen Stil des Repos (siehe [CONTRIBUTING](CONTRIBUTING.md)).
- **Keine Belästigung, kein Doxxing, keine Hassrede.** Gewöhnliche Grundlinie — Issues und PRs, die das verletzen, werden geschlossen und der Account blockiert, ohne Debatte.
- **Off-Topic-Drama wird geschlossen.** Dieses Repo dreht sich um die drei Säulen. Unzusammenhängende Streitereien gehören woanders hin.

Melde ein Problem über ein Issue, oder kontaktiere [@KeilerHirsch](https://github.com/KeilerHirsch) direkt für alles, das du nicht öffentlich posten willst.
